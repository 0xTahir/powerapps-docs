# SortStatus

[!INCLUDE [sort-description](includes/sortstatus-description.md)]

## name

The name of the column

**Type**: `string`

## sortDirection

<!-- ColumnSortDirection  -->
The current sort direction for the column.

**Type**: `enum`

The `sortDirection` value is an enum with the following possible values

|Value|Member|
|--|--|
|-1|None|
|0|Ascending|
|1|Descending|
