# Metadata

## DisplayName

**Type**: `string`

## LogicalName 

**Type**: `string`

## IsSecured

**Type**: `boolean`

## SourceType

**Type**: `number`

## Description

**Type**: `string`

## RequiredLevel

**Type**: `RequiredLevel`

The `RequiredLevel` has following values:

|value|RequiredLevel|
|---|---|
|-1|Unknown|
|0|None|
|1|SystemRequired|
|2|Applicationrequired|
|3|Recommended|
