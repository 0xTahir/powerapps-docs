## Syntax

`formatDateShort(value, includeTime)`

## Parameters

| Parameter Name|Type|Required|Description|
| ------------- |----|--------|-----------|
|value|`Date`|yes|Value Date to format.|
|includeTime|`boolean`|yes|Whether to show time in formatted value.|

## Return Value

Type: `string`
